New
Create a new client using existing cliend/server .dll from the specified Template path.

Basically copies the specified client with .dll files, configs and NOT stuff like .vpk, materials, models, etc...
For instance: You have Half-Life 2 engine with hl2, episodic, ep2 (e.g. ...\steamapps\common\Half-Life 2\hl2) and you want to create a standalone (with custom models only n shit) mod based off ep2.
If so - choose ep2 as a base and a new client will be created which is essentially the same as ep2, but with no maps, models and other content.

The content which is being copied is:
  • Gameinfo (links like episodic and hl2 remain)
  • Configs
  • PBR, if any
  • Resources
